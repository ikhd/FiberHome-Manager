## 🌟 Overview

FiberHome Manager is a **professional Windows desktop application** for advanced users who need **full visibility and control** over FiberHome LTE / 5G routers.

It provides:  
📡 Real-time signal analytics  
📶 Carrier Aggregation monitoring  
🏗️ Neighbor cell inspection  
⚙️ Advanced network control tools  

> ⚠️ The project is distributed as a standalone EXE and is **not open source**.

---

## 💾 Installation

1. Download the latest **EXE** from the **Releases** section.  
2. Run the application.  
3. No additional installation required. ✅
